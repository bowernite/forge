# Forge CLI

A TypeScript CLI tool for repository validation and service management.

## Quick Start

1. **Install dependencies:**

   ```bash
   bun install
   ```

2. **Build the executable:**

   ```bash
   bun run build
   ```

3. **Install globally (optional):**
   ```bash
   bun run global-install
   ```
   This creates a symlink, so that any programs/binaries are available globally in your shell.
